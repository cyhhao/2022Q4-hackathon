## 作品简介
Lumi，一项为上帝信徒的代祷和捐赠带来团结的协议。
在 Lumi dApp 中，信徒的祈祷将以代币的形式呈现，可以促进和捐赠给慈善项目。
在一些慈善项目中提供帮助的用户将获得 SBT 徽章。他们可以使用徽章找到他们的社区。
