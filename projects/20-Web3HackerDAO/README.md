## 作品简介


Web3VitePress 基于 VitePress 技术基础上，添加了 Web3 相关的 UX/UI， 目标是能让 VitePress 用户可以在上面写小说或者其他书时，可以把小说或者书发布为 NFT，其读者可以通过 mint NFT 来获得书的不同章节的阅读权限。

此次将使用 《元宇宙赏金黑客传奇》小说的共创场景来展示整个产品平台的功能。
